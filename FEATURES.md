# HomePage 功能清单 / Features List

## 最新更新 / Latest Updates

### 🎨 预设主题色 / Preset Theme Colors
- 8种精美配色方案可快速切换
- 包括薰衣草紫、樱花粉、天空蓝、清新绿、温暖橙、梦幻紫、热情红、青绿色
- 可视化调色板，点击即可应用
- 自动高亮当前使用的配色方案

### 📊 顶部小组件栏 / Top Widgets Bar
- 所有小组件移至顶部横向排列，节省空间
- 紧凑设计，信息一目了然
- 支持水平滚动查看更多小组件
- 包含：
  - 🕐 时钟：显示当前时间和日期
  - 🌤 天气：显示温度、天气状况、湿度和风速
  - ✓ 待办：显示待办事项数量
  - 📝 笔记：显示笔记预览（前30字符）

### 👁️ 小组件可见性控制 / Widget Visibility Controls
- 在设置面板中可独立控制每个小组件的显示/隐藏
- 自定义您的工作台，只显示需要的信息
- 设置实时生效，状态持久保存

### 🖱️ 增强鼠标特效 / Enhanced Cursor Effects
- 更大的光标跟随点（16px，原为8px）
- 更多的点击粒子效果（12个，原为8个）
- 增强的动画效果：
  - 光标脉冲动画范围更大（1.8倍，原为1.5倍）
  - 粒子扩散距离更远（40-70px，原为30-50px）
  - 粒子带有发光效果
- 更长的动画持续时间（800ms，原为600ms）

### 🖼️ 高级背景控制 / Advanced Background Controls

#### 显示模式 / Display Modes
- **覆盖（Cover）**：保持比例，覆盖整个背景
- **适应（Contain）**：保持比例，完整显示图片
- **填充（Fill）**：拉伸填充，可能变形
- **自定义（Custom）**：完全自定义大小和位置

#### 自定义选项 / Custom Options（仅在自定义模式）
- 大小控制：10% - 200%
- 水平偏移：0% - 100%
- 垂直偏移：0% - 100%

#### 视觉效果 / Visual Effects
- **模糊强度**：0 - 20px，营造磨砂玻璃效果
- **不透明度**：10% - 100%，控制背景透明度
- **饱和度**：0% - 200%，调整色彩鲜艳程度

### 🌸 随机动漫壁纸增强 / Enhanced Anime Wallpaper
- 支持多个API端点，提高成功率：
  1. waifu.pics
  2. waifu.im
  3. nekos.best
- 自动故障转移，一个失败立即尝试下一个
- 更可靠的壁纸获取体验

## 原有功能 / Original Features

### 🎯 快捷方式 / Shortcuts
- 可拖拽排序
- 分组管理
- 自定义图标和链接

### ✅ 待办事项 / Todo List
- 添加、完成、删除待办
- 实时统计待办数量
- 本地存储，永不丢失

### 📝 便签笔记 / Notes
- 自动保存（800ms防抖）
- 大容量文本编辑
- 实时同步到顶部小组件预览

### 🌤️ 天气信息 / Weather
- 基于wttr.in API
- 显示温度、体感温度
- 湿度和风速信息

### 🔖 书签收藏 / Bookmarks
- 快速添加常用网站
- 独立于快捷方式的书签管理

### 🕐 历史记录 / Recent History
- 自动记录最近访问的12个网站
- 一键清除

### 🎨 主题切换 / Theme Toggle
- 亮色/暗色主题
- 自动适配系统主题
- 平滑过渡动画

### 🔍 搜索引擎 / Search Engines
- Google、百度、Bing、DuckDuckGo
- 快速切换搜索引擎
- 支持键盘快捷键

### 💾 数据持久化 / Data Persistence
- 所有数据存储在本地LocalStorage
- 隐私优先，无需注册登录
- 无追踪，无上传

## 技术特性 / Technical Features

- 纯原生JavaScript，无框架依赖
- 响应式设计，支持桌面/平板/移动端
- CSS动画和过渡效果
- 模块化代码结构
- 完整的错误处理
- 性能优化（防抖、节流）

- Chrome/Edge 90+
- Firefox 88+
- Safari 14+
- 移动端浏览器
